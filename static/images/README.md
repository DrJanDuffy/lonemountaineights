# Images

Site images organized by purpose.

## Folders

- **icons/** – Favicon, app icons, UI icons, social icons
- **agents/** – Agent and team headshots (e.g., Dr. Jan Duffy)
- **photos/** – Property photos, neighborhood shots, content imagery
- **hero/** – Hero/background images for landing pages
- **og/** – Open Graph and social share images

Reference in code as `/images/path/to/file.jpg` (served from `static/`).
